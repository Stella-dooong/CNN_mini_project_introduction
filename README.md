# NumPy CNN五分类图像识别

本项目是课程实践作业，**完全基于NumPy**实现卷积神经网络，完成**bird、car、cat、dog、person**五类图像的识别任务。项目未使用PyTorch、TensorFlow等任何深度学习框架，最终在Eval AI测试集上达到93%的准确率。

---

## 数据集

使用导师提供的小规模五分类图像数据集：

| 类别 | 训练集 | 验证集 |
|------|--------|--------|
| bird | 100张 | 50张 |
| car | 100张 | 50张 |
| cat | 100张 | 50张 |
| dog | 100张 | 50张 |
| person | 100张 | 50张 |

- **训练集总计**：500张图片
- **验证集总计**：250张图片
- **输入格式**：100×100 灰度图，像素值归一化到 [0,1]
- **数据量较小**，保留全尺寸输入以避免关键视觉特征丢失

---

## 基线模型

项目初始版本为纯NumPy实现CNN，基础结构：
Conv → ReLU → MaxPool → Conv → ReLU → MaxPool → Flatten → FC → ReLU → FC 

**基线主要问题**：
- 训练稳定性不足
- 小样本下容易过拟合
- 原始卷积计算效率较低


---

## 优化方案

### 1. 模型结构优化

| 优化项 | 基线 | 优化后 | 效果 |
|--------|------|--------|------|
| 激活函数 | ReLU | **LeakyReLU** | 减少神经元死亡，梯度传播更稳定 |
| 权重初始化 | 标准随机 | **He Initialization** | 参数分布适配ReLU类激活函数 |
| 优化器 | Mini-batch GD | **Mini-batch GD with Momentum (β=0.9)** | 降低训练震荡，加快收敛 |

### 2. 训练策略优化

| 优化项 | 说明 |
|--------|------|
| 输入尺寸 | 保留 **100×100** 全尺寸 |
| 训练轮数 | 调整为 **25 epochs** |
| 数据增强 | 随机水平翻转 + 旋转 (±10°) |

### 3. 计算效率优化

将原始卷积中的多层Python循环改为NumPy向量化计算，通过滑动窗口和矩阵运算减少重复计算。
